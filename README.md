# Portfolio Website — Irena Daneva

You can visit on https://danevairena.github.io/PortfolioWebsite-React-Bootstrap/

---

## About this project  
This is my personal portfolio website designed to showcase my background, skills, and projects in a clean and modern way. It was built using React with React Bootstrap and is fully responsive, aiming to reflect my creative side along with my technical foundation.

---

## My Role & Goals  
As a UI/UX-oriented front-end developer, I created the entire website—from conceptual design to full implementation. My goals were:
- To craft a visually appealing and cohesive experience  
- To ensure the site is responsive and performs well  
- To organize my profile information, skills section and contact details in a user-friendly layout  

---

## Tech Stack  
- React.js  
- React Bootstrap  
- HTML5 & CSS3  
- GitHub Pages (deployment)

---

## Key Features  
- Responsive multi-section layout (Home, About Me, Education, Skills, Contact)  
- Smooth navigation and clear visual hierarchy  
- Reusable React components for sections and UI blocks  
- Simple and clean design suitable for a creative technical professional
